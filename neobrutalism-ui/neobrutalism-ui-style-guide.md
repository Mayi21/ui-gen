# UI设计风格规范 - 新野兽主义风格（Neobrutalism）

## 一、整体风格定位
采用新野兽主义设计风格，强调大胆、直接、趣味化的视觉表达。通过粗黑边框、高饱和度色彩和扁平化布局，营造现代、年轻、友好的用户体验。

## 二、色彩系统

### 主色调
- **黄色（Primary）**: #FFD670 / RGB(255, 214, 112) - 用于主要操作按钮、筛选状态
- **青绿色（Secondary）**: #4ECDC4 / RGB(78, 205, 196) - 用于生成、确认等正向操作
- **洋红色（Danger）**: #FF6B9D / RGB(255, 107, 157) - 用于删除、取消等危险操作
- **蓝色（Processing）**: #6BB6FF / RGB(107, 182, 255) - 用于进行中状态
- **绿色（Success）**: #5FD068 / RGB(95, 208, 104) - 用于完成状态

### 中性色
- **纯黑色（Border/Text）**: #000000 / RGB(0, 0, 0) - 边框和主文本
- **暖白色（Background）**: #F8F5F2 / RGB(248, 245, 242) - 页面背景
- **白色（Card）**: #FFFFFF / RGB(255, 255, 255) - 卡片、表格背景
- **深灰色（Disabled）**: #9E9E9E / RGB(158, 158, 158) - 禁用状态

### 色彩原则
- 饱和度保持在70-90%之间
- 避免渐变，使用纯色块
- 同一界面限制3-4种主题色
- 确保文本与背景对比度≥4.5:1

### 色彩应用场景
- **黄色 #FFD670**：主要操作、默认筛选、启动任务
- **青绿色 #4ECDC4**：生成、确认、正向反馈
- **洋红色 #FF6B9D**：危险操作、删除、取消、强调选中
- **蓝色 #6BB6FF**：进行中状态、信息提示
- **绿色 #5FD068**：成功状态、完成标识
- **红色 #FF5252**：错误、失败状态
- **黑色 #000000**：所有边框、主要文字
- **灰色 #9E9E9E**：次要文字、禁用状态

## 三、边框与描边

### 边框规范
- **粗细**: 3px（标准）/ 4px（强调）/ 2px（小元素）
- **颜色**: 纯黑 #000000
- **应用**: 所有交互元素（按钮、卡片、输入框、表格）必须有边框

### 分割线
- 使用3px纯黑实线
- 避免使用虚线或点线（文件上传等特殊场景除外）

## 四、圆角规范

### 圆角半径
- **大型容器**: 24-32px（卡片、对话框、大面板）
- **中型元素**: 16-20px（按钮、输入框）
- **小型元素**: 8-12px（标签、小按钮）
- **状态标签**: 24px（椭圆形）

### 圆角原则
- 保持一致性，同类元素使用相同圆角
- 避免完全直角，保持友好感

## 五、按钮设计

### 主要按钮（Primary）
- 背景色：黄色 #FFD670
- 边框：3px 纯黑 #000000
- 圆角：20px
- 内边距：14px 28px
- 文字：16px 加粗，黑色 #000000
- Hover：背景色加深10%，边框可选加粗至4px
- Active：向右下偏移2px（立体按压效果）

### 次要按钮（Secondary）
- 背景色：青绿色 #4ECDC4
- 边框：3px 纯黑 #000000
- 其他规范同主要按钮

### 危险按钮（Danger）
- 背景色：洋红色 #FF6B9D
- 边框：3px 纯黑 #000000
- 其他规范同主要按钮

### 文本按钮（Text/Ghost）
- 背景色：透明或白色
- 边框：3px 纯黑 #000000
- 文字：黑色 #000000
- Hover：浅灰色背景 #F5F5F5

### 按钮组/筛选按钮（Button Group/Filter）
- 单个按钮：椭圆形，圆角24px
- 边框：3px 纯黑 #000000
- 内边距：10px 24px
- 间距：12px
- 未选中：白色 #FFFFFF 背景
- 选中：默认使用黄色 #FFD670，可根据场景切换其他主题色
- 数字徽章：显示在文字后，加括号，如"全部 (50)"

### 禁用状态
- 背景色：#E0E0E0
- 边框颜色：#9E9E9E
- 文字颜色：#9E9E9E
- 禁用光标交互

## 十二、状态标识

### 状态标签（Badge）
- 圆角：24px（椭圆形）
- 边框：2px 纯黑 #000000
- 内边距：8px 20px
- 字体：14px 加粗，黑色 #000000

### 状态颜色
- 运行中：蓝色 #6BB6FF 背景
- 已完成：绿色 #5FD068 背景
- 失败/错误：红色 #FF5252 背景
- 待处理/排队中：灰色 #E0E0E0 背景
- 警告：黄色 #FFD670 背景

## 十三、排版与间距

### 字体规范
- 主标题：24-32px，加粗
- 副标题：18-20px，中等
- 正文：16px，常规
- 辅助文字：14px，常规
- 字体族：优先使用系统字体（-apple-system, BlinkMacSystemFont, "Segoe UI", Roboto...）

### 间距系统
- 基础单位：4px
- 组件内边距：8px, 12px, 16px, 24px
- 组件外边距：16px, 24px, 32px
- 栅格间距：16px

## 十四、图标设计

### 图标规范
- 风格：线性或填充，保持统一
- 描边粗细：2px
- 颜色：纯黑 #000000 或主题色
- 尺寸：16px（小）, 20px（中）, 24px（大）, 32px（超大）
- 圆角：轻微圆角（2px）
- 交互图标：可添加3px纯黑边框的圆形/方形背景

## 十五、响应式适配

### 移动端调整
- 边框缩减至2px
- 圆角适当减小（8-12px）
- 按钮内边距：10px 20px
- 触摸区域最小44×44px
- 表格采用卡片式布局

### 桌面端
- 保持标准规范
- 适当增加留白
- 支持Hover状态

## 十六、动效原则

### 动画时长
- 快速交互：150ms
- 标准交互：250ms
- 复杂动画：350ms

### 缓动函数
- ease-out（出场动画）
- ease-in-out（状态切换）

### 可用动效
- 按钮按压：transform translate
- 状态切换：背景色过渡
- 避免过度动画，保持简洁

## 十七、可访问性

### 对比度
- 文本对比度≥4.5:1
- 大文本对比度≥3:1

### 焦点状态
- 使用明显的焦点边框
- 焦点边框宽度：3px
- 焦点颜色：蓝色 #0066FF

### 语义化
- 使用语义化HTML标签
- 提供aria标签
- 支持键盘导航
