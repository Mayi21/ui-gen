# UI设计风格规范 - 新野兽主义风格（Neobrutalism）

## 一、整体风格定位
采用新野兽主义设计风格，强调大胆、直接、趣味化的视觉表达。通过粗黑边框、高饱和度色彩和扁平化布局，营造现代、年轻、友好的用户体验。

## 二、色彩系统

### 主色调
- **黄色（Primary）**: #FFD670 / RGB(255, 214, 112) - 用于主要操作按钮、筛选状态
- **青绿色（Secondary）**: #4ECDC4 / RGB(78, 205, 196) - 用于生成、确认等正向操作
- **洋红色（Danger）**: #FF6B9D / RGB(255, 107, 157) - 用于删除、取消等危险操作
- **蓝色（Processing）**: #6BB6FF / RGB(107, 182, 255) - 用于进行中状态
- **绿色（Success）**: #5FD068 / RGB(95, 208, 104) - 用于完成状态

### 中性色
- **纯黑色（Border/Text）**: #000000 / RGB(0, 0, 0) - 边框和主文本
- **暖白色（Background）**: #F8F5F2 / RGB(248, 245, 242) - 页面背景
- **白色（Card）**: #FFFFFF / RGB(255, 255, 255) - 卡片、表格背景
- **深灰色（Disabled）**: #9E9E9E / RGB(158, 158, 158) - 禁用状态

### 色彩原则
- 饱和度保持在70-90%之间
- 避免渐变，使用纯色块
- 同一界面限制3-4种主题色
- 确保文本与背景对比度≥4.5:1

### 色彩应用场景
- **黄色 #FFD670**：主要操作、默认筛选、启动任务
- **青绿色 #4ECDC4**：生成、确认、正向反馈
- **洋红色 #FF6B9D**：危险操作、删除、取消、强调选中
- **蓝色 #6BB6FF**：进行中状态、信息提示
- **绿色 #5FD068**：成功状态、完成标识
- **红色 #FF5252**：错误、失败状态
- **黑色 #000000**：所有边框、主要文字
- **灰色 #9E9E9E**：次要文字、禁用状态

## 三、边框与描边

### 边框规范
- **粗细**: 3px（标准）/ 4px（强调）/ 2px（小元素）
- **颜色**: 纯黑 #000000
- **应用**: 所有交互元素（按钮、卡片、输入框、表格）必须有边框

### 分割线
- 使用3px纯黑实线
- 避免使用虚线或点线（文件上传等特殊场景除外）

## 四、圆角规范

### 圆角半径
- **大型容器**: 24-32px（卡片、对话框、大面板）
- **中型元素**: 16-20px（按钮、输入框）
- **小型元素**: 8-12px（标签、小按钮）
- **状态标签**: 24px（椭圆形）

### 圆角原则
- 保持一致性，同类元素使用相同圆角
- 避免完全直角，保持友好感

- **中型**：32×32px（按钮内加载）
- **小型**：24×24px（行内加载）

#### 样式
- 形状：圆形
- 边框宽度：4px
- 边框颜色：
  - 静态部分（75%）：纯黑 #000000
  - 旋转部分（25%）：青绿色 #4ECDC4
- 圆角：完全圆形（rounded-full）

#### 动画
- 名称：`animate-neo-spin`
- 时长：1s
- 缓动函数：ease-in-out
- 循环：无限（infinite）
- 旋转：360度

#### 可选文字
- 位置：加载器下方
- 间距：16px
- 字体：14px 中等
- 颜色：灰色 #9E9E9E
- 内容："加载中..." / "Loading..."

#### CSS 实现
```css
@keyframes neo-spin {
  from { transform: rotate(0deg); }
  to { transform: rotate(360deg); }
}

.animate-neo-spin {
  animation: neo-spin 1s ease-in-out infinite;
}
```

## 十五、响应式适配

### 移动端调整（< 768px）

#### 边框调整
- 标准边框：3px → 2px
- 强调边框：4px → 3px
- 分割线：3px → 2px

#### 圆角调整
- neo-xl (24px) → neo-lg (20px)
- neo-lg (20px) → neo-md (12px)
- neo (16px) → neo-md (12px)
- neo-md (12px) → neo-sm (8px)
- neo-pill (24px) → 保持不变

#### 按钮调整
- 内边距：px-7 py-3.5 → px-5 py-2.5
- 圆角：neo-lg (20px) → neo-md (12px)
- 字体：16px → 14px（可选）
- 边框：3px → 2px

#### 卡片调整
- 内边距：24px → 16px
- 圆角：neo-xl (24px) → neo-lg (20px)
- 边框：3px → 2px

#### 输入框调整
- 内边距：14px 16px → 10px 12px
- 圆角：neo (16px) → neo-md (12px)
- 边框：3px → 2px

#### 标签调整
- 内边距：8px 20px → 6px 16px
- 圆角：neo-pill (24px) → 20px
- 边框：2px → 2px（保持）

#### 触摸区域
- 所有交互元素最小尺寸：44×44px
- 适用于：
  - 按钮
  - 链接
  - 复选框
  - 单选框
  - 开关按钮
  - 下拉选择器

#### 布局调整
- 表格：改为卡片式堆叠布局
- 多列网格：改为单列
- 侧边栏：收起或改为顶部导航
- 模态框：全屏显示

### 平板端（768px - 1024px）
- 保持桌面端样式
- 适当缩小间距
- 网格列数：2-3列

### 桌面端（> 1024px）
- 保持标准规范
- 适当增加留白
- 支持完整 Hover 状态
- 支持键盘导航
- 显示工具提示

## 十六、动效原则

### 动画时长
- 快速交互：150ms
- 标准交互：250ms
- 复杂动画：350ms

### 缓动函数
- ease-out（出场动画）

### 可用动效
- 按钮按压：transform translate
- 状态切换：背景色过渡
- 避免过度动画，保持简洁
